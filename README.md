# coffee-shop-sales-dashboard-

Project Overview

This project analyzes coffee shop sales performance using an interactive dashboard. The dashboard provides insights into sales trends, order volume, product performance, store performance, and customer purchasing patterns.

Key Metrics
Total Sales: $82K
Total Orders: 17,314
Total Quantity Sold: 24,870
Dashboard Features
Monthly Sales Analysis
Sales Trend Over Time
Product Category Performance
Weekday vs Weekend Sales
Store Location Performance
Hourly Sales Heatmap
KPI Tracking with Month-over-Month Comparison
Tools Used
Power BI
Microsoft Excel
DAX
Data Visualization
Business Insights
Identified top-performing product categories.
Analyzed peak sales hours.
Compared weekday and weekend sales.
Evaluated store-level performance.
Monitored sales growth trends.
